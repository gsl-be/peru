# Peru Map Web App

To install
```
git clone https://github.com/gsl-be/peru.git
cd peru
npm install 
```

Run the app

`node app.js`

View the app
http://localhost:3000


Your home page is `views/index.html` file. It contains a map of Peru from the google maps api. https://developers.google.com/maps/documentation/javascript/tutorial
